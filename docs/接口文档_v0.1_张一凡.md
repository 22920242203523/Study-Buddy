2. 作文批改接口（核心接口）
请求地址：
POST http://127.0.0.1:8000/api/composition/correct
请求头：
textContent-Type: application/json
请求参数：
JSON{
  "content": "作文正文（支持自然换行）",
  "grade": "中年级"          // 可选，默认值：中年级
}
grade 可选值：低年级 / 中年级 / 高年级
请求示例：
JSON{
  "content": "那束光，从未熄灭\n暮色四合，窗外的雨丝斜斜地织着。我坐在书桌前，看着那张皱巴巴的英语试卷，心里很难过。\n妈妈端着一杯热牛奶走进来，轻声说：“别着急，慢慢来。”",
  "grade": "高年级"
}

返回参数（成功时）：
JSON{
  "total_score": 82,
  "dimension_scores": {
    "内容与主题": "23/25",
    "结构与逻辑": "21/25",
    "语言表达": "20/25",
    "创新与情感": "18/25"
  },
  "strengths": [
    "主题明确，围绕‘那束光’展开，寓意深刻。",
    "语言简洁流畅，富有画面感和情感色彩。"
  ],
  "weaknesses": [
    "情节略显单薄，缺乏具体事例支撑。"
  ],
  "suggestions": "可以加入更多细节描写，如学习过程中的具体困难或突破，使文章更生动真实。",
  "inspiring_questions": [
    "你是否曾在某个瞬间感受到内心的‘那束光’？它带给你怎样的力量？",
    "如果让你用一个比喻来形容‘坚持’，你会选择什么？为什么？"
  ],
  "encouragement": "你的作文充满情感与思考，虽然还有提升空间，但已经展现了良好的写作潜力。继续努力，相信你会写出更精彩的作品！"
}
返回字段说明：

total_score：总分（0-100）
dimension_scores：四维度评分（当前均为满分25分）
strengths：优点列表（数组）
weaknesses：不足列表（数组）
suggestions：改进建议（字符串）
inspiring_questions：启发问题列表（3个左右）
encouragement：鼓励语


3. 健康检查接口
请求地址：
GET http://127.0.0.1:8000/health
返回示例：
JSON{
  "status": "ok",
  "message": "中小学生学习助手后端服务正常运行"
}

4. 本地测试地址

5. 注意事项（重要）

作文内容支持自然换行，无需手动输入 \n。
开发阶段前端可直接调用 http://127.0.0.1:8000。
后期部署后会提供正式服务器地址。
如果返回 total_score 为 75 分且 suggestions 为“系统正在优化中”，说明后端处理异常，请及时告诉我。
建议前端在调用时增加错误处理和加载状态提示。
