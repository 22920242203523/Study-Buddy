# FastAPI 学习笔记（张一凡 2026-03-24）

## 1. 为什么选择 FastAPI？
- 性能高（基于 Starlette 和 Pydantic）
- 支持异步（async/await），适合调用 LLM API
- 自动生成 Swagger 交互文档（/docs）
- 类型提示友好，代码更安全
- 非常适合构建 AI 后端服务

## 2. 核心概念学习总结（官方教程前3节重点）

- **FastAPI 实例**：`app = FastAPI()`
- **路径操作装饰器**：`@app.get()`、`@app.post()`
- **Pydantic 模型**：用于定义请求体和响应体（数据验证）
- **异步函数**：`async def`（推荐用于调用外部 API）
- **依赖注入**：`Depends()`（后续可用于权限、数据库等）

## 3. 项目中计划使用的后端架构

**整体流程**：
用户请求 → FastAPI 路由 → Prompt 组装 → 调用 LLM API（Qwen） → 解析返回 → 格式化输出 → 返回给前端（Flutter）
