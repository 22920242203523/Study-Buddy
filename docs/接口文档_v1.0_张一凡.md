## 2. 作文批改接口（核心接口）

**请求地址**：  
`POST http://10.27.194.72:8000/api/composition/correct`

**请求头**：
Content-Type: application/json

**请求参数**：

```json
{
  "content": "作文正文（支持自然换行）",
  "grade": "中年级",
  "student_id": "student_001"        // 建议传入，用于区分学生
}
grade 可选值：低年级 / 中年级 / 高年级

请求示例：

{
  "content": "那束光，从未熄灭\n暮色四合，窗外的雨丝斜斜地织着。我坐在书桌前，看着那张皱巴巴的英语试卷，心里很难过。\n妈妈端着一杯热牛奶走进来，轻声说：“别着急，慢慢来。”",
  "grade": "高年级",
  "student_id": "zhangyf_001"
}

返回参数（成功时）：
{
  "total_score": 82,
  "dimension_scores": {
    "内容与主题": "23/25",
    "结构与逻辑": "21/25",
    "语言表达": "20/25",
    "创新与情感": "18/25"
  },
  "strengths": ["优点1", "优点2"],
  "weaknesses": ["不足1"],
  "suggestions": "改进建议内容...",
  "inspiring_questions": ["问题1", "问题2", "问题3"],
  "encouragement": "鼓励语..."
}

3. 批改历史查询接口（v1.0 新增）

请求地址：
GET http://10.27.194.72:8000/api/history/{student_id}?limit=10


示例：
http://10.27.194.72:8000/api/history/zhangyf_001?limit=5

返回示例：
[
  {
    "id": 1,
    "student_id": "zhangyf_001",
    "content_preview": "那束光，从未熄灭...",
    "total_score": 82,
    "created_at": "2026-04-17 14:23:45"
  }
]

6. 注意事项（重要）

服务启动命令（局域网访问）：PowerShell    uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

浏览器提示“不安全”时，请点击 高级 → 继续前往。
student_id 建议前端传入，用于区分不同学生和保存历史记录。
作文内容支持直接换行，无需手动写 \n。
已开启 CORS，支持前端跨域调用。
