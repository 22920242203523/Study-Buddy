# 数据库设计文档

**项目名称**：中小学生学习助手  
**文档作者**：张一凡  
**文档版本**：v0.1  
**更新日期**：2026.5.8  
**交付物**：数据库设计 Schema

---

## 1. 设计目标

- 存储学生作文及AI批改记录
- 支持作文批改历史查询
- 支持后续数据统计分析（按年级、分数段、时间等）
- 方便与前端进行数据交互

---

## 2. 核心数据表设计

### 2.1 students（学生表）

```sql
CREATE TABLE students (
    id SERIAL PRIMARY KEY,
    student_id VARCHAR(50) UNIQUE NOT NULL,     -- 学号或系统ID
    name VARCHAR(100),
    grade VARCHAR(20),                          -- 低年级 / 中年级 / 高年级
    class_name VARCHAR(50),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);


2.2 compositions（作文表）
CREATE TABLE compositions (
    id SERIAL PRIMARY KEY,
    student_id INTEGER REFERENCES students(id),
    content TEXT NOT NULL,                      -- 作文原文
    title VARCHAR(200),                         -- 作文标题（可选）
    grade VARCHAR(20),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

2.3 corrections（批改记录表 - 核心表）
CREATE TABLE corrections (
    id SERIAL PRIMARY KEY,
    composition_id INTEGER REFERENCES compositions(id),
    student_id INTEGER REFERENCES students(id),
    
    total_score INTEGER NOT NULL CHECK (total_score BETWEEN 0 AND 100),
    
    -- 四维度评分（25分制）
    score_content INTEGER,      
    score_structure INTEGER,
    score_language INTEGER,
    score_innovation INTEGER,
    
    strengths TEXT[],                           -- 优点列表
    weaknesses TEXT[],                          -- 不足列表
    suggestions TEXT,                           -- 改进建议
    inspiring_questions TEXT[],                 -- 启发问题（数组）
    encouragement TEXT,                         -- 鼓励语
    
    model_used VARCHAR(50) DEFAULT 'qwen-turbo',
    response_time FLOAT,                        -- API响应时间（秒）
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

3. 索引设计（提升查询性能）

CREATE INDEX idx_corrections_student ON corrections(student_id);
CREATE INDEX idx_corrections_composition ON corrections(composition_id);
CREATE INDEX idx_corrections_created ON corrections(created_at DESC);
CREATE INDEX idx_students_grade ON students(grade);



4. 主要业务场景支持

查询某个学生的所有批改历史
查询某篇作文的所有批改记录
按年级统计平均分
查看最近批改记录
支持作文内容搜索（可后续增加全文索引）

5. 后续可扩展方向

用户认证表（auth_users）
批改反馈表（correction_feedback）
